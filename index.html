Javascript Snippets

The following are descriptions of the JavaScript programs included

Fibonnachi1 is a JavaScript program that gives the fibonacci sequence of a number which is the number that is derived from the sum of the two previous numbers. Milestokilos is a JavaScript program that converts miles to kilometers.

Areaperiphery is a JavaScript program that calculates the area or the periphery of a circle given the radius as input by the user. Pyth is a JavaScript program that finds the hypotenuse of a triangle given the other two sides. Finally we have rpsc which is the popular game of rock, paper, scissors.
